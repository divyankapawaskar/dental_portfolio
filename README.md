# Divyanka Pawaskar — Personal Website for Dental School Applications

A single-page portfolio for dental school applications, with content from your CV and a SALUTO-inspired design using your five colors: **#5f2c3e**, **#c2d2bd**, **#d1adcc**, **#c65032**, **#f6e9d7**.

## View locally

- **Option 1:** Open `index.html` in your browser.
- **Option 2:** From this folder run:
  ```bash
  npx serve .
  ```
  Then open **http://localhost:3000** (or the URL shown in the terminal).

## Deploy

- **GitHub Pages:** Push this folder to a repo → Settings → Pages → Source: main branch.
- **Netlify:** Drag this folder to [app.netlify.com/drop](https://app.netlify.com/drop) or connect your repo.

## Project contents

- `index.html` — All sections (Hero, About, Experience, Education, Research, Skills, Contact)
- `styles.css` — Five-color palette, SALUTO-style layout (rose/sage backgrounds, burgundy blocks, cream nav)
- `script.js` — Mobile menu toggle
- `Divyanka_Pawaskar_CV_Sept2025.pdf` — Linked from the Contact section “Download CV” button
